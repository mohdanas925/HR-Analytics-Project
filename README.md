## 📌 Project Overview

The **HR Analytics Dashboard** is an end-to-end Business Intelligence project built using **Power BI** and **SQL**. The project focuses on analyzing employee data to uncover insights into workforce demographics, employee attrition, job satisfaction, and organizational trends.

Unlike a standard dashboard project, this project also includes **SQL-based validation** where every key metric and visualization in the dashboard is verified using SQL queries. This ensures the accuracy and reliability of the dashboard calculations.

This project demonstrates skills in **SQL, Power BI, Data Modeling, DAX, Data Validation, and Business Intelligence**.

# 🎯 Project Objectives

The main objectives of this project are to:

- Analyze employee attrition across the organization.
- Monitor workforce demographics.
- Evaluate job satisfaction across different job roles.
- Understand employee distribution by age and education.
- Build an interactive HR dashboard.
- Validate every dashboard metric using SQL queries.
- Provide HR managers with actionable insights for better decision-making.

# 📈 Dashboard Features

## 1. Employee Summary

Displays overall workforce information including:

- Total Employees
- Active Employees
- Employees Left
- Attrition Rate
- Average Age

---

## 2. Department-wise Attrition

Analyzes employee attrition across departments such as:

- Research & Development
- Sales
- Human Resources

**Business Insight**

Helps identify departments experiencing the highest employee turnover.

---

## 3. Employee Distribution by Age

Displays employees grouped by:

- Under 25
- 25–34
- 35–44
- 45–54
- Above 55

- Female Employees

## 4. Job Satisfaction Analysis

Displays satisfaction ratings (1–4) for each job role.

## 6. Gender-wise Attrition Analysis

Analyzes attrition by:

This helps identify demographic trends in employee turnover.

# 🧮 SQL Validation

One of the major highlights of this project is **SQL validation**.

Every KPI shown in the Power BI dashboard has been validated using SQL queries.

Examples include:

- Total No. of Employees
- Attrition Count
- Active Employees
- Attrition Rate
- Average Age
- Department-wise Attrition Count and Attrition Rate
- Age Group Distribution
- Education-wise Attrition Count and Attrition Rate
- Age bands wise Attrition Count

This practice ensures:

- Dashboard accuracy
- Reliable reporting
- Data consistency
- Cross-verification between SQL and Power BI
